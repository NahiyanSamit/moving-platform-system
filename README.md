# Moving Platform System

UPM package scaffold for a simple moving platform system.

Install locally in Unity by adding the package path to the project's `manifest.json`:

{
  "dependencies": {
    "com.nahiyansamit.moving-platform-system": "file:Packages/com.nahiyansamit.moving-platform-system"
  }
}

Structure:
- Runtime: runtime scripts
- Editor: editor tooling
- Documentation~: package documentation
- Tests: test assemblies
